eBay Connection

Your Client ID is safe here — it's stored locally and only used to build the eBay login URL.

📋 Setup Guide

  1. Go to developer.ebay.com → sign in with your eBay seller account.
  2. Create an application (or use your existing one). Set Application Type to Confidential.
  3. Under OAuth → User Tokens → Redirect URIs, add your Netlify site URL exactly:
    https://YOUR-SITE.netlify.app/
  4. Copy your Client ID and paste it above.
  5. In Netlify → Site → Environment Variables, add:
    EBAY_CLIENT_ID = your Client ID
    EBAY_CLIENT_SECRET = your Client Secret
  6. Deploy this project to Netlify (drag the folder, or connect GitHub).
  7. Click Connect eBay Account above and authorise.

🚀 Deploy to Netlify

  1. Download the ZIP file provided with this app.
  2. Go to netlify.com → Add new site → Deploy manually.
  3. Drag and drop the project folder into Netlify.
  4. Your site URL will appear — use it as your eBay Redirect URI (Step 3 above).
  5. Set environment variables under Site → Environment variables.
  6. Trigger a redeploy so the function picks up the new vars.
